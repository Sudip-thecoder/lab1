CSS -> Cascading Stylesheet

What is CSS?

Document ending with *.css extension or used by declaring a <style> tag whether inside the html file or inside a line to style it and make it look better is called css.

Types of CSS:

  • External Css
  • Internal Css
  • Inline Css
Codes
Inline #Head part ...
#body part --

<p style="color:blue;"> This is a inline styling </p>
Internal #Head part ...
#body part --

<style>
#span1{
 color: red;
}
#span2{
 font-size: 1rem;
}
<style>
<p>
This is how color changes from internal css.
And this is for font.
</p>

External #Head part --
<link rel="stylesheet" href="style.css"/>
#body part --
<p>
This was styled using a external css. Same as the external css file used to style this whole page itself.
</p>

Css is usually written in external file to maintain the readability of html code and any linking of external css or the declaration of <style> tag is usually done inside the <head> tag since it is a good practice to do so.
